## Entity

Minimal Next.js + TypeScript + Tailwind CSS project with a shadcn-style structure.

- **Components root**: `components`
- **UI components**: `components/ui`
- **Global styles**: `app/globals.css`

### LiquidEffectAnimation

- Component file: `components/ui/liquid-effect-animation.tsx`
- Demo page: `app/demo/page.tsx` (visit `/demo` when running `next dev`)

### Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the dev server:

   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000/demo` to see the liquid effect animation.
